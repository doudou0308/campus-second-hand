# 🎓 校园二手交易平台

应急管理大学校园二手交易平台 —— 专为在校师生打造的闲置物品流转网站。

这是我的第一个网站项目，使用原生 HTML + CSS + JavaScript 构建。

---

## 📁 项目结构

```
├── css/
│   └── style.css          # 外部样式表
├── js/
│   └── script.js           # JavaScript 交互脚本
├── img/
│   ├── 663-1920x350.jpg    # 首页轮播图1
│   ├── 866-1920x350.jpg    # 首页轮播图2
│   └── 微信图片_20260422091401_455_40.jpg  # 首页轮播图3
├── index.html              # 首页
├── publish.html            # 发布闲置页面
├── category.html           # 分类浏览页面
├── notice.html             # 交易须知页面
├── contact.html            # 联系我们页面
├── about.html              # 关于我们页面
├── faq.html                # 常见问题页面
├── my-posts.html           # 我的发布页面
└── README.md               # 项目说明文档
```

## 🌐 页面说明

| 页面 | 文件名 | 说明 |
|------|--------|------|
| 首页 | `index.html` | 平台入口，展示热门分类、热门商品、最新发布 |
| 发布闲置 | `publish.html` | 发布二手物品信息，含表单验证 |
| 分类浏览 | `category.html` | 按类别筛选浏览商品，支持交互式分类切换 |
| 交易须知 | `notice.html` | 交易流程、安全指南、违规物品清单 |
| 联系我们 | `contact.html` | 用户留言表单，含邮箱格式验证 |
| 关于我们 | `about.html` | 平台介绍与特色说明 |
| 常见问题 | `faq.html` | FAQ 问答列表 |
| 我的发布 | `my-posts.html` | 查看和管理已发布的物品 |

## ✨ 技术特点

- **纯原生技术**：HTML5 + CSS3 + 原生 JavaScript，无框架依赖
- **响应式布局**：Flexbox + Grid 弹性布局，适配手机、平板、电脑
- **外部样式**：CSS 独立为 `style.css`，统一管理所有页面样式
- **交互功能**：
  - 首页轮播图点击切换
  - 分类浏览实时筛选
  - 表单验证（必填项、邮箱格式、价格合法性）
  - 搜索框支持回车键
  - 回到顶部按钮
- **语义化标签**：`header`、`section`、`footer`、`aside` 等 HTML5 语义标签

## 🚀 如何运行

1. 克隆或下载本项目到本地
2. 用浏览器直接打开 `index.html` 即可浏览
3. 也可使用 Live Server 等工具启动本地服务器
